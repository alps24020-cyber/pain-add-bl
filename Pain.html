Tutorial Buat GHP GitHub

01.

Masuk ke GitHub, klik foto profil > Settings.

02.

Pilih menu paling bawah: Developer Settings.

03.

Klik Personal access tokens > Tokens (classic).

04.

Klik Generate, centang kotak "repo" (Wajib agar script bisa menulis file).

05.

Klik Generate Token, copy kodenya dan simpan di catatan.
